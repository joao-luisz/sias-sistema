# SIAS - Sistema Integrado de Atendimento Social

Sistema moderno de gestão de filas e atendimento para a Secretaria de Assistência Social.

## Funcionalidades

- **Recepção:** Emissão de senhas, cadastro de cidadãos e triagem.
- **Painel de TV:** Chamada de senhas com voz e alertas visuais.
- **Atendente:** Gestão de filas, atendimento e histórico.
- **Admin:** Gestão de serviços e usuários.
- **Relatórios:** Métricas de desempenho, horários de pico e exportação CSV.

## Tecnologias

- React + Vite
- TypeScript
- Tailwind CSS
- Supabase (Auth, Database, Realtime, Storage)
- Google Gemini AI (Resumo de Relatórios)

## Como Rodar

1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure o `.env` com suas chaves do Supabase.
4. Rode o projeto: `npm run dev`.
